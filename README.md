# Kudoboard
## Kudoboard is an appreciation platform for special occasions, events, shoutouts, etc. 
## Create a digital card (kudoboard) and invite other people to add their contributions. Then deliver the appreciation.
### Tech Stach Used: 
#### Frontend:
* Next Js
* React Query
* MUI
#### Backend:
* Express Js
* MongoDB
* Moongose
* JWT
